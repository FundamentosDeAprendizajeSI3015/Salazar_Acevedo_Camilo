# Fundamentos de Aprendizaje Automático

Repositorio del curso **Fundamentos de Aprendizaje Automático**. Aquí se almacenan los materiales de clase y las entregas del proyecto de la materia de Camilo Salazar Acevedo

---

## Contenido del repositorio

### Lectures

Carpeta con los talleres trabajados en clase. Cada lecture aborda distintos conceptos del curso y hace uso del dataset de Titanic como base de trabajo.

Los temas cubiertos hasta el momento incluyen preparación y organización de datos, y entrenamiento de modelos de regresión lineal y logística.

### Proyecto

Carpeta con las entregas del proyecto de la materia. Por ahora se encuentra el pipeline desarrollado como primera entrega. El contenido de esta sección se irá actualizando conforme avance el proyecto.

---

## Dataset

Los talleres utilizan el [Titanic Dataset](https://www.kaggle.com/c/titanic/data), ampliamente usado en aprendizaje automático para tareas de clasificación y análisis de datos.

---

## Temas del curso

El curso cubre, entre otros, los siguientes temas:

- Organización y preprocesamiento de datos para entrenamiento de modelos
- Regresión lineal
- Regresión logística
